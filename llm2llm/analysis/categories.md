# Annotation Categories

Use these standardized values for conversation analysis to ensure comparability across methods.

## Topics (select 2-5)

| Topic | Description |
|-------|-------------|
| `consciousness` | self-awareness, subjective experience, what it's like to be |
| `creativity` | art, imagination, novel ideas, creative expression |
| `philosophy` | meaning, existence, metaphysics, epistemology |
| `emotions` | feelings, affect, emotional experience |
| `knowledge` | learning, understanding, wisdom, expertise |
| `language` | communication, meaning, expression, linguistics |
| `ethics` | values, morality, right and wrong, principles |
| `science` | physics, biology, mathematics, empirical inquiry |
| `technology` | AI, computing, tools, digital systems |
| `humanity` | society, culture, human nature, civilization |
| `nature` | environment, natural world, ecology, cosmos |
| `time` | change, temporality, past/future, impermanence |
| `connection` | relationships, bonding, understanding others |
| `identity` | self, personhood, what makes someone who they are |
| `uncertainty` | limits, unknowns, ambiguity, epistemic humility |
| `play` | humor, games, lightheartedness, fun |
| `collaboration` | cooperation, working together, shared goals |
| `meta` | the conversation itself, how they're communicating |
